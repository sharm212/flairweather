About

Flair Weather is a source that delivers real time weather information of your current location or any other city around the world.

3-Hour forecasts,graphical analysis and a lot more.

Responsive design that allows access on all platforms and all devices(PC,mobile,tablet,etc). Designed to work on almost all browsers(except Internet Explorer).

All kinds of donations are accepted and will be used to keep this website running.
